# Jarvis Local LLM Assistant 🤖

A Jarvis-inspired AI voice assistant built in Python using Ollama for local LLM-based responses.  
This project listens to user voice commands, processes them, and responds using speech output.

## 🚀 Features

- 🎤 Voice input using speech recognition
- 🗣️ Text-to-speech response generation
- 🧠 Local LLM integration with Ollama
- 💬 Short chat history for contextual responses
- ⚡ Lightweight and fast local execution
- 🔒 Runs locally for better privacy and control

## 🛠️ Tech Stack

- Python
- Ollama
- SpeechRecognition
- pyttsx3 / edge-tts
- requests
- python-dotenv
